# AYP Tech Website - Modern Technology Consulting

A modern, futuristic website for AYP Tech - a technology consulting company that helps non-technical businesses integrate technology to scale their operations.

## 🚀 Features

### Design Elements
- **Modern Futuristic UI**: Dark theme with neon accents (blue/purple)
- **Glassmorphism Effects**: Transparent cards with blur effects
- **Smooth Animations**: Micro-animations and scroll effects
- **Responsive Design**: Mobile-first approach
- **Tech-inspired Graphics**: Animated grid backgrounds and floating elements

### Pages Included
1. **Homepage** (`index.html`) - Complete landing page with all sections
2. **About Us** (`about.html`) - Company story, mission, values, and team

## 🔧 Customization

### Adding New Services
1. Add service card in the services grid
2. Create detailed section with benefits
3. Update navigation and footer links
4. Add to contact form dropdown

### Modifying Colors
Update CSS custom properties in `:root` selector:
```css
:root {
    --primary-color: #your-color;
    --secondary-color: #your-color;
    /* etc. */
}
```

### Adding Animations
Use the existing animation classes or create new ones:
- `.glass-card` for glassmorphism effects
- `.gradient-text` for gradient text
- `.neon-text` for glowing text effects
